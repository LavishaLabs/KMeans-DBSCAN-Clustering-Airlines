# K-Means and DBSCAN Clustering on Airlines Dataset
## Objective
Perform clustering analysis using K-Means and DBSCAN algorithms on airline
customer data.
## Techniques Used
- Data Preprocessing
- Exploratory Data Analysis
- K-Means Clustering
DBSCAN Clustering
- Silhouette Score Evaluation
## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
## Results
K-Means produced compact customer segments, while DBSCAN effectively identified
noise and irregular patterns.
